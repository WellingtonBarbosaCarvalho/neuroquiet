# GlucoTrust Product Images

This folder contains the product images for GlucoTrust (German version).

## Required Images:

- `PRODx1-500px.webp` - 1 bottle image
- `PRODx3-500px.webp` - 3 bottles image
- `PRODx6-500px.webp` - 6 bottles image

## Image Specifications:

- Format: WebP
- Size: 500px (optimized for web)
- Alt text should include: "GlucoTrust" + package size + benefit description

## Usage:

These images are referenced in `glucotrust.html` for the pricing cards.
